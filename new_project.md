1. Create a new project (new folder)
2. `uv init` - initiate project files
3. `uv venv` - create Virtual Environment and activate it
   - `source .venv/bin/activate` on MacOS
   - `.venv/Scripts/activate.ps1` (or `.bat` when using Command Prompt)

4. `uv pip install numpy` - install `numpy`
5. Open the project in VSCode, create a new file that imports numpy and the file should run.
